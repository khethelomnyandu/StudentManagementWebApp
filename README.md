# Student Management System

A simple application to manage student information efficiently.  
This project allows you to add, update, view, and delete student records, helping streamline basic administrative tasks for educational institutions.  
Ideal as a learning project for practicing C# and database management concepts.

---

## Features

- Add, edit, and delete student records
- Search students by name or ID
- Display student details in a structured format
- Simple and user-friendly interface

---

## Technologies Used

- C#,JavaScript
- .NET 10.0 
- SQL Server
